רימון uses yt-dlp — the best free video downloader — to grab this video directly to your computer. No ads, no tracking, yours forever. One-time setup, then it's just one command every time.
① Install yt-dlp
Open Command Prompt (search it in Start menu) and run:
winget install yt-dlp
No winget? Download yt-dlp.exe directly from github.com/yt-dlp/yt-dlp/releases and put it in your Downloads folder.
② Install Node.js (prevents extraction errors)
Required to avoid YouTube extraction warnings:
winget install OpenJS.NodeJS.LTS
③ Install ffmpeg (required for 1080p and above)
Without ffmpeg you're limited to 720p. With it you get 1080p, 4K, and best audio:
winget install ffmpeg
④ Verify everything is installed
Restart your terminal, then check each one:
node -v
ffmpeg -version
Both should print a version number. If they do, you're ready.
⑤ Download this video
Run this command — replace the URL with the one below:
yt-dlp --js-runtimes node "https://youtube.com/watch?v=VIDEO_ID"
For MP4 format (instead of WebM), add the format flag:
yt-dlp --js-runtimes node -f "bv[ext=mp4]+ba[ext=m4a]/b[ext=mp4]" "https://youtube.com/watch?v=VIDEO_ID"
That's it. Once installed, you only ever need the yt-dlp command. It downloads to whatever folder your terminal is currently in. Files are yours — no DRM, play them in VLC or any video player.
רימון משתמש ב־yt-dlp — המוריד הטוב ביותר בחינם — כדי להוריד את הסרטון ישירות למחשב שלך. בלי פרסומות, בלי מעקב, שלך לתמיד. הגדרה חד פעמית, ואז רק פקודה אחת בכל פעם.
① התקן את yt-dlp
פתח שורת פקודה (חפש "Command Prompt" בתפריט Start) והרץ:
winget install yt-dlp
אין winget? הורד את yt-dlp.exe ישירות מ־github.com/yt-dlp/yt-dlp/releases ושים אותו בתיקיית Downloads.
② התקן Node.js (מונע שגיאות חילוץ)
נדרש כדי למנוע אזהרות של YouTube:
winget install OpenJS.NodeJS.LTS
③ התקן ffmpeg (חובה ל־1080p ומעלה)
בלי ffmpeg אתה מוגבל ל־720p. עם ffmpeg תקבל 1080p, 4K ואודיו מיטבי:
winget install ffmpeg
④ בדוק שהכל מותקן
הפעל מחדש את שורת הפקודה, ואז בדוק:
node -v
ffmpeg -version
שתיהן אמורות להציג מספר גרסה. אם כן — אתה מוכן.
⑤ הורד את הסרטון הזה
הרץ את הפקודה הזו — הקישור כבר מוכן:
yt-dlp --js-runtimes node "https://youtube.com/watch?v=VIDEO_ID"
לקובץ MP4 (במקום WebM), הוסף את הדגל:
yt-dlp --js-runtimes node -f "bv[ext=mp4]+ba[ext=m4a]/b[ext=mp4]" "https://youtube.com/watch?v=VIDEO_ID"
זהו. אחרי ההתקנה, אתה צריך רק את הפקודה yt-dlp. הקבצים יורדים לתיקייה שבה שורת הפקודה נמצאת כרגע. הם שלך — ללא DRM, פתח אותם ב־VLC או בכל נגן וידאו.
