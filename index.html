Welcome to GoaTube
Connect your free YouTube API key to search and stream millions of real videos. Takes about 5 minutes.
1
Go to console.cloud.google.com and sign in with Google.
2
Click New Project → any name → Create.
3
Enable YouTube Data API v3.
4
Credentials → Create → API Key. Copy it.
5
Paste it below and click Connect.
Free: 10,000 requests/day. Stored locally in your browser.
